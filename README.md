# 開始使用
1. 先確認有安裝 node.js 與 npm
2. 將專案下載至本地電腦
3. 透過終端機進入資料夾，輸入： npm install
4. 安裝完成後在終端機輸入： npm run dev
5. 看到以下訊息表示成功開啟： Welcome to http://localhost:3000
6. 打開瀏覽器輸入以下網址： http://localhost:3000
7. 關閉伺服器連線請在終端機輸入： ctrl+c

# 開發工具
* Node.js 14.16.0 
* Express 4.17.3
* Express-Handlebars 6.0.3
